# Implementation Plan: AI Intelligence Layer

**Branch**: `004-ai-intelligence-layer` | **Date**: 2026-04-25 | **Spec**: [spec.md](spec.md)  
**Input**: Feature specification from `specs/004-ai-intelligence-layer/spec.md`

---

## Summary

Transform the Personal OS from a pull-based, module-centric system into a proactive, question-centric operating system. Six phases ship independently. Phase 0 activates proactive Telegram in 15 minutes (no code). Phase 5 adds full conversational AI via Telegram. Phases 1–4 redesign the home page, decision system, trade-off visibility, and weekly review. All execution layer components (Routine, Schedule, Quick Capture, existing Telegram commands) are untouched.

**Pre-condition**: `core/services.py` (754 lines, limit 400) must be split before Phase 4 adds to it. This is a Constitution IX gate — see Complexity Tracking.

---

## Technical Context

**Performance Goals**: Phase 1 home page must render with zero additional API calls vs. current  
**Constraints**: Render free tier (cold start ~50s); no new Python packages without justification; all migrations must run cleanly on PostgreSQL  
**Scale/Scope**: Single active user (Mohamed); architecture supports small trusted group

## Phase 5 — Telegram Conversational Mode

**Pre-conditions**: None (one migration, isolated to core app).

**Backend**:
- `core/models.py` — add `TelegramConversation` model (migration: `NNNN_telegramconversation.py`)
- `core/telegram_bot.py` — modify `handle_webhook()`: route non-command text to `_handle_conversation()`; create `_handle_conversation(chat_id, text)` function; create `_format_actions(actions)` helper
- `core/chat_service.py` — already has `"telegram"` mode added in Phase 1

**Execution flow in `_handle_conversation`**:
1. Load `TelegramConversation.get_recent(chat_id, limit=10)`
2. Append user message
3. Call `run_chat(messages, context={"mode": "telegram"})`
4. Save exchange via `TelegramConversation.append(chat_id, text, result["reply"])`
5. Format reply: if `result["actions"]`, append `"\n\n✓ {_format_actions(result['actions'])}"`
6. Call `send_message(chat_id, formatted_reply)`

**Constitution VIII**: `telegram_bot.py` gets updated bilingual comments on new functions.
